Sobre o projeto

Um projeto de portfólio com problema real, evolução incremental e foco em usabilidade.

O Simulador de Escala de Trabalho nasceu como um projeto Python de linha de comando para praticar fundamentos importantes: datas, funções, validações, JSON, testes e organização de código.

A evolução natural foi transformar a lógica em uma demo web e, depois, preparar a base para um sistema mais profissional com POO, Service, Repository e PostgreSQL.

O projeto cresceu por versões, sem pular etapas.
